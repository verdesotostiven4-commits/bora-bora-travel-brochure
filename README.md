# Bora Bora Travel Brochure

Interactive mini presentation and brochure for the English task **“The Perfect Tourist Experience”**.

## What this project includes

- A creative travel agency landing page.
- A 6-slide interactive presentation for a 2–3 minute oral report.
- A tri-fold inspired brochure section with all required panels.
- Tourism vocabulary highlighted in each section.
- Gerunds and infinitives used naturally in context.
- CSS illustrated image and drawing areas.
- Travel recommendations.
- Creative extras: live package price builder, passport stamps, presentation mode, keyboard controls, and print brochure option.

## Grammar included

### Gerunds

- enjoy visiting
- love exploring
- suggest staying
- avoid traveling
- prefer backpacking
- snorkeling
- swimming

### Infinitives

- want to visit
- decide to stay
- plan to travel
- hope to see
- need to book

## How to present

1. Open `index.html` in the browser.
2. Click **Start the Trip**.
3. Use **Presentation Mode** for a cleaner view.
4. Use the **Next** button or keyboard arrows.
5. Turn on **Highlight Grammar** to show gerunds and infinitives clearly.
6. Use the **Build Your Dream Package** and **Passport Stamps** sections as the creative part.

## Recommended Pages setup

Go to the repository settings, open **Pages**, choose **Deploy from branch**, select `main`, and save. Then the static site will be published.
